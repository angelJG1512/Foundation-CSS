¿Qué es un Framework CSS?

Un framework CSS es una biblioteca preescrita que está destinada a permitir un diseño web más fácil y que cumpla con los estándares web. Proporciona una base sólida para construir aplicaciones web responsivas y estéticamente atractivas.
